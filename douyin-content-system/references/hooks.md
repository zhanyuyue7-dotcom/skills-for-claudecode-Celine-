# 开头与结尾模板库

## 开头钩子

以下5种开头模板可交替使用。对于5-10分钟的长口播，开头不需要像短视频那样"3秒定生死"——可以用30-60秒从容地把观众带进来。关键不是快，是**让人觉得接下来的话值得听**。模板是骨架，实际使用时应该用自然、口语化的方式展开，像在跟一个朋友聊天。

### 钩子A：反直觉炸弹（涨粉型）

**结构**：抛出一个违反常识的断言 → 停顿 → 展开

**模板**：
> "[违反常识的结论]。[停顿]。我知道你可能不同意，但听我说完。"

**示例**：
- "大一学生不应该学编程语言。"
- "AI不会取代程序员，但会取代不会用AI的程序员。"
- "我花了两个月做了一个可能永远卖不出去的东西，但我觉得值。"

**风格参考**：Levelsio — 用内部人视角说出反直觉真相

### 钩子B：事件驱动（蹭热点型）

**结构**：[时间词] + [主体] + [发生了什么] → 我试了一下 → 结果...

**模板**：
> "[昨天/刚刚/今天凌晨]，[公司/产品]发布了[XX]，我第一时间试了一下，[一句话结论]。"

**示例**：
- "昨天 Claude 更新了一个新功能，我试完之后代码效率直接翻倍。"
- "OpenAI 刚发布了 Codex，我用它跑了一下我的项目，结果出乎意料。"

**风格参考**：Riley Brown — 新闻事件 + 第一人称实测

### 钩子C：过程展示（建信任型）

**结构**：这是我做XX的第N天 + 今天的进展/翻车

**模板**：
> "这是我做[项目名]的第[N]天。今天，[具体进展或翻车]。"

**示例**：
- "这是我做桌面贾维斯的第47天。今天它终于能听懂我的语音指令了。"
- "这是我做桌面贾维斯的第52天。今天机械臂把我的杯子甩飞了。"

**风格参考**：Karpathy — 真实过程记录，不修饰

### 钩子D：数字筛人（精准获客型）

**结构**：一句话说出目标人群 + 具体数字/场景 → 如果你是这种人，接着看

**模板**：
> "如果你[具体场景/身份]，但是[痛点]，这条视频就是给你的。"

**示例**：
- "如果你是学生，想用AI做点东西但不知道从哪开始，这条视频就是给你的。"
- "如果你已经用了三个月的ChatGPT但还是只会聊天，听我说两分钟。"

**风格参考**：MFM (Sam Parr) — 第一句话筛选观众

### 钩子E：提问式（引发思考型）

**结构**：抛出一个让人想回答的问题 → 停顿 → 给出意想不到的切入角度

**模板**：
> "[一个好问题]？[停顿]。大多数人会说[常见答案]，但真正的答案是[意外角度]。"

**示例**：
- "你觉得AI最大的能力是什么？不是写代码，不是画图，是让一个大一学生能做出原来要一个团队才能做的东西。"
- "具身智能到底是什么？不是机器人，是让AI长出手脚。"

## 结尾模板库（柴静式收尾）

结尾是从"有用"到"有意义"的升华。好的结尾让观众记住你这个人，而不只是记住内容。

### 结尾A：回归人性

**结构**：从具体技术/项目拉回到人的层面

**模板**：
> "说到底，[技术/工具/项目]只是手段。真正重要的是[人性层面的洞察]。"

**示例**：
- "说到底，AI只是工具。真正稀缺的，是你想用它来解决什么问题的那个想法。"
- "这个机械臂还很笨拙，但每次它成功抓起一个东西，我都觉得——人造出来的东西，正在学着理解这个世界。"

### 结尾B：未完成的故事

**结构**：留一个悬念或进行中的状态，让人想跟下去

**模板**：
> "[事情还没结束/问题还没解决]。[我会继续/下一步是XX]。[留一个小悬念]。"

**示例**：
- "贾维斯现在能听、能看、能动了。但它还不会'想'。这是下一步。"
- "这个bug我还没完全搞定。但我有了一个新思路，明天试。"

### 结尾C：金句定格

**结构**：用一句高度浓缩的话收束全文，像电影最后一帧

**模板**：一句独立的、有节奏感的短句。

**示例**：
- "未来不会等你准备好。"
- "最好的学习方式，就是造一个你造不出来的东西。"
- "工具在进化，问题是——你在进化吗？"

### 结尾D：自我对话

**结构**：从讲给观众听，转为讲给自己听

**模板**：
> "其实我今天说的这些，也是在提醒我自己——[内心独白]。"

**示例**：
- "其实我今天说的这些，也是在提醒我自己——别光想，去做。哪怕做出来的东西很烂。"
- "我有时候也会怀疑，一个大一的做这些有什么用。但转念一想——不做才是真的没用。"
